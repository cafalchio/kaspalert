# kaspalert

It will play You Suffer by Napalm Death when kaspa reaches the treshold (current 0.2 USD)

 * git clone "this repo"
 * Create an environment: virtualenv .venv
 * Enable de env source .venv/bin/activate
 * cd kaspalert
 * pip install -r requirements
 * python main.py
